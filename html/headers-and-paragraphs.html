Headers and Paragraphs

This page will explain the use of headers and paragraphs.

Headers

Headers are a good way to call attention to certain things on your webpage. This helps the user navigate your site quickly by scanning for the things that are of interest to them.

We can set various attributes for the header, thus giving it prominence through its size, color and text decoration.

Paragraphs

Paragraphs are the main way we display text, so they should be clean and easy-to-read.

Blockquote

How about a long quote from someone? Perhaps in Latin to review your grade school classical training? This could go in a blockquote:

Bold and Italics

If something is super important, you might want to emphasize it using italics. Or you might want to make it bold so that it really stands out. If you are going for serious text overkill, you could even use both (but that would be really dumb).
