Head and Body

This section will explain the various parts of an html page.

Head

The head is the place where we put all the "secret" or private information pertaining to the page. Here are some things that would go in the <head> section:

  • Document type
  • Page title
  • The link to the style sheet (Or, if using in-line style, the style code would appear in this section as well (e.g., for html emails).)
  • The character set required

Body

Footer

Sometimes footers are used at the bottom of a page. These can contain additional information including: a sitemap, contact information, a company logo and other exciting things.
